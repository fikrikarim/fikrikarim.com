My eyes hurt when I stare at a screen for too long. There are a lot of break reminder apps out there, and a popular one, LookAway, costs $38. There are also free apps, but most of them don't have automatic meetings detection, which is a key requirement for me.

With the cost of creating simple software approaching thanks to Claude Code (if you don't count the cost of the 20x Max subscription haha), I decided to build my own break reminder app that's totally free and open source. It only took me a few hours.

Here's the repo to download the app: https://github.com/fikrikarim/repose

TIL it's super easy to setup automatic updates for macOS apps using [Sparkle](https://github.com/sparkle-project/Sparkle).

Fun fact: The logo is generated from an SVG.
